# pocock-engineering

Engineering lifecycle skills forked from Matt Pocock's framework, adapted for the Dancer marketplace.

## Skills

| Skill | Purpose |
|---|---|
| `setup-matt-pocock-skills` | Scaffold per-repo config (issue tracker, triage labels, domain docs). Run first. |
| `tdd` | Test-driven development with red-green-refactor loop |
| `to-issues` | Break a plan/PRD into independently-grabbable issues |
| `to-prd` | Turn conversation context into a PRD |
| `triage` | Issue triage state machine |
| `diagnose` | Disciplined diagnosis loop for hard bugs |
| `improve-codebase-architecture` | Find deepening opportunities using CONTEXT.md + ADRs |
| `zoom-out` | Zoom out for broader context |

## Attribution

These skills originate from [Matt Pocock's skill framework](https://github.com/anthropics/claude-code-plugins), forked and adapted by Jack Reis. The original framework is MIT-licensed.

## Installation

```bash
# Via Dancer marketplace
/plugin install pocock-engineering@dancer
```

## First Use

Run `/setup-matt-pocock-skills` to configure your repo's issue tracker, triage labels, and domain docs before using the other skills.
